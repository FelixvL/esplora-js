# esplora.js
JavaScript Client for Blockstream's Esplora

See [Blockstream Esplora API Documentation](https://github.com/Blockstream/esplora/blob/master/API.md)

Features: 
  - Mainnet & Testnet
  - Error Handling
  - JSDocs
  - No Dependencies
  - Compatible with tree shaking
  - Error Handling API
  - most endpoints implemented
