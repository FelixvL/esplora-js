# esplora.js
JavaScript Client for Blockstream's Esplora. 

## Demo
[You can find the demo here](https://coins.github.io/esplora.js/demo.html)


## Features: 
  - Mainnet & Testnet
  - Error Handling
  - JSDocs
  - No Dependencies
  - Compatible with tree shaking
  - Error Handling API
  - most endpoints implemented
